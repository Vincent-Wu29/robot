### 2. C++开发环境搭建
- 编译器选择：
  - Windows: MinGW, Visual Studio
  - Linux: g++ (GNU C++编译器)
  - Mac: clang++ 或 g++
- 安装验证：
  ```bash
  g++ --version
  ```
- 编译运行流程：
  1. 编写源代码 (.cpp文件)
  2. 编译：`g++ hello.cpp -o hello`
  3. 运行：`./hello` (Linux/Mac) 或 `hello.exe` (Windows)

### 3. C++基础语法
- 程序结构：
  ```cpp
  #include <iostream>
  int main() {
      std::cout << "Hello World" << std::endl;
      return 0;
  }
  ```
- 变量与数据类型：
  - 基本类型：`int`, `double`, `char`, `bool`
  - 字符串：`std::string` (需要`#include <string>`)
  - 常量：`const`关键字
- 输入输出：
  - 输出：`std::cout << value`
  - 输入：`std::cin >> variable`
  - 换行：`std::endl` 或 `"\n"`
- 运算符：算术、关系、逻辑运算符与Python类似

### 4. 控制结构
- 条件语句：
  ```cpp
  if (condition) {
      // 代码块
  } else if (condition) {
      // 代码块
  } else {
      // 代码块
  }
  ```
- 循环语句：
  - `for`循环：`for (int i = 0; i < n; i++)`
  - `while`循环：`while (condition)`
  - `do-while`循环：至少执行一次
- 与Python的主要差异：
  - 条件必须用括号`()`
  - 代码块必须用花括号`{}`
  - 缩进不是语法要求（但建议保持）

### 5. 函数
- 函数定义：
  ```cpp
  返回类型 函数名(参数列表) {
      // 函数体
      return 返回值;
  }
  ```
- 函数声明与定义可以分开
- 函数重载：同一函数名，不同参数列表
- 与Python函数的主要差异：
  - 必须指定返回类型
  - 参数必须指定类型
  - 支持函数重载（Python不支持）

## 📊 C++与Python关键差异总结

| 特性 | C++ | Python |
|------|-----|--------|
| 类型系统 | 静态类型，编译时检查 | 动态类型，运行时检查 |
| 内存管理 | 手动/智能指针 | 自动垃圾回收 |
| 执行方式 | 编译执行 | 解释执行 |
| 性能 | 高 | 较低 |
| 开发速度 | 较慢 | 快 |
| 语法严谨性 | 高，错误在编译时发现 | 较低，错误在运行时发现 |
| 学习曲线 | 陡峭 | 平缓 |
| 适用领域 | 系统编程、游戏、嵌入式 | Web、数据分析、AI |
| 机器人应用 | 底层控制、实时系统 | 上层应用、算法原型 |
