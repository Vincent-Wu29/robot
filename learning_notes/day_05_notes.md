### 1. 文件操作基础
- 文件打开模式：`'r'`(读)、`'w'`(写)、`'a'`(追加)、`'b'`(二进制)
- 文件操作步骤：
  1. 打开文件：`open(filename, mode, encoding)`
  2. 读取/写入数据
  3. 关闭文件：`close()`或使用`with`语句自动管理
- 读取方法：
  - `read()`：读取全部内容
  - `readline()`：读取一行
  - `readlines()`：读取所有行到列表
- 写入方法：
  - `write(string)`：写入字符串
  - `writelines(list)`：写入字符串列表

### 2. 异常处理
- 异常处理结构：
  ```python
  try:
      # 可能引发异常的代码
  except SpecificError:
      # 处理特定异常
  except (Error1, Error2):
      # 处理多个异常
  except Exception as e:
      # 处理所有其他异常
  else:
      # 没有异常发生时执行
  finally:
      # 无论是否异常都会执行
  ```
- 常见异常类型：
  - `ValueError`：值错误
  - `TypeError`：类型错误
  - `FileNotFoundError`：文件不存在
  - `ZeroDivisionError`：除零错误
  - `KeyError`：字典键不存在
  - `IndexError`：索引越界
- 抛出异常：`raise Exception("错误信息")`
- 自定义异常类

### 3. 模块与包
- 模块导入方式：
  - `import module_name`
  - `import module_name as alias`
  - `from module_name import function_name`
  - `from module_name import *`（不推荐）
- `__name__`变量的作用：
  - 当模块直接运行时，`__name__`等于`"__main__"`
  - 当模块被导入时，`__name__`等于模块名
- 模块化设计的好处：
  - 代码复用
  - 易于维护
  - 分工协作
- 包（package）结构：
  ```
  mypackage/
  ├── __init__.py
  ├── module1.py
  └── subpackage/
      ├── __init__.py
      └── module2.py
  ```

### 5. LeetCode第14题：最长公共前缀
- 题目理解：
- 解法1：水平扫描法
  ```python
  def longestCommonPrefix(strs):
      if not strs:
          return ""

      prefix = strs[0]
      for s in strs[1:]:
          while not s.startswith(prefix):
              prefix = prefix[:-1]
              if not prefix:
                  return ""
      return prefix
  ```
- 解法2：垂直扫描法
  ```python
  def longestCommonPrefix(strs):
      if not strs:
          return ""

      for i in range(len(strs[0])):
          char = strs[0][i]
          for s in strs[1:]:
              if i >= len(s) or s[i] != char:
                  return strs[0][:i]
      return strs[0]
  ```
- 时间复杂度分析：O(S)，其中S是所有字符串字符总数
- 空间复杂度：O(1)
