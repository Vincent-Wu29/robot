### 1. 函数进阶：可变参数与关键字参数
- *args的理解与应用：
  - 接收任意数量的位置参数
  - 在函数内部作为元组使用
  - 示例：`def sum_all(*args)`

- **kwargs的理解与应用：
  - 接收任意数量的关键字参数
  - 在函数内部作为字典使用
  - 示例：`def print_info(**kwargs)`

- 参数组合顺序：
  ```
  def func(a, b=1, *args, **kwargs)
  ```

### 2. 递归函数
- 递归三要素：
  1. 基本情况（base case）
  2. 递归情况（recursive case）
  3. 向基本情况推进

- 阶乘函数实现：
  ```python
  def factorial(n):
      if n == 0 or n == 1:
          return 1
      else:
          return n * factorial(n-1)
  ```

### 3. 列表推导式
- 基础语法：`[expression for item in iterable]`

- 嵌套循环：`[expression for item1 in iterable1 for item2 in iterable2]`
- 优势：代码简洁、执行效率高
